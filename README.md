# Azure-Cloud-Computing-Fundamentals
# Cloud Providers

This document explains the three major cloud providers and what they offer: Microsoft Azure, Amazon Web Services (AWS), and Google Cloud Platform (GCP).

---

## Microsoft Azure
Azure allows businesses to run virtual machines, servers, and applications online without needing physical hardware. Companies can scale their computing power up or down depending on demand.

### Key Features
- Virtual machines and scalable compute
- Data analytics and business intelligence tools
- AI and machine learning services
- Automation and DevOps tools

---

## Amazon Web Services (AWS)
AWS provides a global cloud platform for hosting websites, apps, and virtual machines. It is known for its wide range of services and strong security.

### Key Features
- Global infrastructure for apps and websites
- Secure cloud storage for files and backups
- AI and machine learning tools
- Containers and serverless computing
- Encryption and access control

---

## Google Cloud Platform (GCP)
GCP lets businesses run applications on Google’s global network. It offers reliable storage and scalable computing resources.

### Key Features
- Virtual machines and app hosting
- Secure and scalable storage
- Data analytics and big data tools
- Easy resource scaling


# Cloud Service Models: IaaS, PaaS, SaaS

This document explains the three main cloud service models and how they are used in the real world.

---

## IaaS — Infrastructure as a Service
IaaS allows users to rent virtual machines, storage, and networking without owning physical hardware.

### Real-World Example
Netflix uses IaaS to rent servers and storage from cloud providers around the world, ensuring smooth streaming in all regions.

---

## PaaS — Platform as a Service
PaaS provides a complete environment for developing and running applications without managing servers or operating systems.

### Real-World Example
Search engines like Google use PaaS-style platforms to deploy code globally. The platform automatically handles updates, scaling, and server management.

---

## SaaS — Software as a Service
SaaS delivers software over the internet. Users access applications through a login rather than installing them locally.

### Real-World Example
Microsoft 365 provides cloud‑hosted productivity tools that can be accessed from any device with an internet connection.


# What Can Cloud Computing Do for Us?

Cloud computing powers much of the technology we use every day. Streaming platforms such as Netflix, Spotify, and YouTube rely on the cloud to store content and deliver it to users worldwide.

---

# How Can Cloud Computing Benefit a Business?

Cloud computing helps businesses work faster, scale easily, and reduce costs. Companies can increase storage or computing power whenever needed without buying new hardware.

This is especially useful for:
- Growing businesses
- Seasonal demand spikes
- Remote or distributed teams
- Data-heavy operations

---

# What’s the Alternative to Cloud Computing?

The alternative is **on‑premise computing**, where a business keeps its servers, storage, and software on-site. The company is responsible for maintenance, updates, and security instead of relying on a cloud provider.

# On-Premise vs Cloud Computing

## Cloud Computing
- Hosted by a cloud provider
- Scalable on demand
- Lower upfront cost
- Provider handles updates and security
- Accessible from anywhere

## On-Premise Computing
- Hardware stored in the company’s building
- High upfront cost
- Company manages updates and security
- Limited scalability
- Requires physical maintenance
